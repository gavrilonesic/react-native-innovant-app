## Get Started

Installation with npm

```sh
npm install react-native-innovant
```

Installation with yarn

```sh
yarn add react-native-innovant
```

### Usage

```jsx
import {Container, Button, ThemeProvider, ContextWrapper} from 'react-native-innovant'

export const App = () => {
    return (
        <ThemeProvider>
         <ContextWrapper>
          <Container>
           <Button title="Submit">
           </Container>
          </ContextWrapper>
        </ThemeProvider>
    )
}

```
